## 1장 파이썬 설치와 설정

이 장에서는 파이썬을 설치하고 설정하는 방법에 대해 설명한다. 이미 파이썬 설치하여 사용하고 있는 독자는 이 장을 생략하고 다음 장부터 학습을 시작할 수 있다.

### 파이썬의 버전
파이썬은 다른 언어와 달리 역사적으로 2 버전과 3 버전이 동시에 활발하게 사용되어 왔다. 하지만 2019년을 기점으로 대부분의 패키지와 커뮤니티에서 2 버전 지원을 중단했다. 따라서 이 책에서는 3.x 버전을 기준으로 설명한다. 

### 파이썬 설치
설치의 편의성을 위해서는 아나콘다 배포판을 이용하는 것이 편리하다. 하지만 고급 사용자가 될 수록 여러가지 버전의 파이썬 인터프리터를 동시에 설치해 놓고 필요에 따라 서로 다른 환경에서 파이썬을 사용하는 경우가 발생하므로 여기서는 파이썬 기본 인터프리터 설치, IDE로는 VS code를 사용하는 것을 기준으로 설명한다.

#### 리눅스와 맥OS

(예 - C:\Python\Python312\Scripts\;C:\Python\Python312\)

### 스크립트 모드 프로그래밍
스크립트 매개변수를 입력한 인터프리터를 호출하면 스크립트 실행이 시작되어 스크립트가 완료될 때까지 계속된다. 스크립트가 완료되면 인터프리터는 더 이상 활성화되지 않는다.
간단한 Python 프로그램을 스크립트로 작성해 보자. 파이썬 파일의 확장자는 .py이고 에디터를 이용하여 `test.py` 파일에 다음과 같이 2줄짜리 소스 코드를 입력해보자. 
```
# test.py
print("Hello, Python!")
```
PATH 변수에 Python 인터프리터가 설정되어 있다고 가정한다. 이제 이 프로그램을 다음과 같이 실행해 보자 
```
$ python test.py
```
그러면 다음과 같은 결과가 나온다. 
```
Hello, Python!
```
### 스크립트 모드 프로그래밍 2
리눅스 환경에서 Python 스크립트를 실행하는 다른 방법을 알아본다. 여기 수정된 `test.py` 파일이 있다. 

```
#!/usr/bin/Python
print("Hello, Python!")
```
/usr/bin 디렉터리에 Python 인터프리터가 설치되어 있다고 가정한다. 이제 이 프로그램을 다음과 같이 실행해 보자 
```
$ chmod +x test.py  # 이것은 파일을 실행 가능하게 만들기 위한 것입니다
$./test.py          # test.py 실행
```
그러면 다음과 같은 결과가 나온다. 
```
Hello, Python!
```

## 파이썬 식별자(Identifier)
파이썬 식별자는 변수, 함수, 클래스, 모듈 또는 기타 객체를 식별하는 데 사용되는 이름이다. 식별자는 A에서 Z 또는 a에서 z로 시작하는 글자 또는 밑줄(_)으로 시작하며, 그 뒤에 0개 이상의 글자, 밑줄, 숫자(0부터 9까지)가 이어진다.
파이썬은 식별자 내에 @, $, %와 같은 특수 문자를 허용하지 않는다. 그리고 파이썬은 대소문자를 구분하는 프로그래밍 언어이다. 따라서 Manpower와 manpower는 파이썬에서 서로 다른 두 식별자이다.
다음은 파이썬 식별자에 대한 명명 규칙이다. 
●	클래스 이름은 대문자로 시작한다. 다른 모든 식별자는 소문자로 시작한다.
●	식별자를 하나의 선두 밑줄로 시작하면 그 식별자가 지역적(local)임을 나타낸다.
●	식별자를 두 개의 선두 밑줄로 시작하는 것은 강한 지역 식별자(local identifier)임을 나타낸다.
●	식별자 뒤에 두 개의 밑줄이 붙어 끝날 경우, 식별자는 언어 정의 특수 이름입니다.

### 들여쓰기
Python은 클래스 및 함수 정의나 흐름 제어를 위한 코드 블록을 나타내는 데 괄호를 사용하지 않는다. 대신 코드 블록은 엄격하게 적용되는 들여쓰기로 표시됩니다.
블록 내 모든 문장은 반드시 다음과 같이 동일한 간격으로 들여쓰기 해야 한다. 예를 들어 − 
```py
if True:
   print "True"
else:
   print "False"

```
하지만 다음과 같은 블록은 오류를 발생한다 − 
```py
if True:
print "Answer"
print "True"
else:
print "Answer"
print "False"
```
따라서 파이썬에서는 모든 행들이 같은 간격의 들여쓰기로 블록을 형성한다. 다음 예시는 다양한 코드 블록의 사례이다 

```py
#!/usr/bin/python
import sys

# 1. 파일 이름 입력받기
file_name = input("Enter filename: ")
if len(file_name) == 0:
    print("Next time please enter something")
    sys.exit()

file_finish = "done"  # 종료 키워드 정의
file_text = ""

# 2. 파일 쓰기 작업
try:
    # open file stream (with 구문을 사용하면 close를 자동으로 해줘서 안전합니다)
    with open(file_name, "w", encoding="utf-8") as file:
        print(f"Enter '{file_finish}' When finished")
        
        while file_text != file_finish:
            file_text = input("Enter text: ")
            if file_text == file_finish:
                break
            file.write(file_text)
            file.write("\n")
except OSError:  # Python 3에서는 IOError가 OSError로 통합되었습니다.
    print("There was an error writing to", file_name)
    sys.exit()

```

### 다중 라인 문장
파이썬의 문장은 보통 한 줄로 끝나지만 줄 연속 문자(\)를 사용하여 여러 줄로 입력하는 것도 가능하다. 예를 들어 −
```
total = item_one + \
        item_two + \
        item_three

```
한편 [], {}, ()등 괄호 안에 포함된 문장은 줄 연속 문자를 사용할 필요가 없다. 예를 들어 − 
```
days = ['Monday', 'Tuesday', 'Wednesday',
        'Thursday', 'Friday']
```

### 파이썬 인용문
파이썬은 문자열 리터럴을 나타내기 위해 단일 ('), 더블("), 세 개의 따옴표(''' 또는 """)를 사용할 수 있다. 단, 동일한 유형의 따옴표가 문자열의 시작과 끝에 함께 사용되어야 한다.

삼중 따옴표는 여러 줄에 걸쳐 문자열을 입력하는 데 사용된다. 예를 들어, 다음의 사례들은 모두 사용가능하다. 
```py
word = 'word'
sentence = "This is a sentence."
paragraph = """This is a paragraph. It is
made up of multiple lines and sentences."""
```

### 파이썬 주석
해시 기호(#)로 시작하는 문자열은 주석으로 간주된다. #과 물리적 행 끝까지의 모든 문자는 주석에 포함되며, Python 인터프리터는 이를 무시한다.

```py
#!/usr/bin/python

# First comment

```

## 파이썬을 계산기로 사용하기
가장 간단한 프로그래밍은 파이썬을 계산기로 사용하는 것이다. 여기에 나오는 모든 코드 예제는 주피터 노트북에서 실행하는 것을 가정하고 있다. 하지만 아이파이썬이나 일반 파이썬 콘솔에서도 실행할 수 있다.

### 파이썬의 연산자
파이썬에서는 정수의 덧셈, 뺄셈, 곱셈, 나눗셈 등을 할 수 있다. 하지만 연산 기호는 수학에서 사용하는 것과 조금 다를 수 있다. 특히 나눗셈 연산은 두 가지 종류가 있다는 점에 주의한다.

* <kbd>+</kbd> : 더하기  
* <kbd>-</kbd> : 빼기 
* <kbd>*</kbd> : 곱하기 
* <kbd>/</kbd> : 나누기 (소수점까지 나누기) 
* <kbd>/</kbd><kbd>/</kbd> : 몫을 구하기
* <kbd>%</kbd> : 나머지 구하기 
* <kbd>**</kbd> : 제곱 

```
2 + 4 - 5
# out[] : 1
2 * 4
# out[] : 8
11 // 4  # 11을 4로 나눈 몫은 2
# out[] : 2
11 % 4  # 11을 4로 나눈 나머지는 3
# out[] : 3
10 / 4  # 10/4 = 2.5
# out[] : 2.5
2 ** 3  # 2의 세제곱은 8
# out[] : 8
```

### 부등식의 참과 거짓 계산
파이썬은 참과 거짓을 계산하는 부등식 연산도 할 수 있다. 파이썬에서는 참과 거짓을 `True` 또는 `False` 라는 값으로 나타낸다. 등호 기호는 변수에 값을 할당하는데 사용되므로 비교 연산을 할 때는 등호 기호가 두 개 반복되는 기호를 사용한다는 점에 주의한다.

* <kbd>></kbd> : 보다 크다 
* <kbd><</kbd> : 보다 작다 
* <kbd>=</kbd><kbd>=</kbd> : 같다 
* <kbd>!</kbd><kbd>=</kbd> : 같지 않다 
* <kbd>></kbd><kbd>=</kbd> : 보다 크거나 같다 
* <kbd><</kbd><kbd>=</kbd> : 보다 작거나 같다

```py
2 > 1
# out[] : True
2 == 1
# out[] : False
3 <= 3
# out[] : True
```

### 부울리언 대수
파이썬에서는 참과 거짓에 대해 <kbd>&</kbd> (AND) 라는 연산과 <kbd>|</kbd> (OR) 라는 부울리언 대수(Boolean algebra) 연산도 할 수 있다. 

* <kbd>&</kbd> (AND) 연산은 두 값이 모두 참일 때만 답이 참이 된다. 즉, 하나라도 거짓이면 답은 거짓이다.
* <kbd>|</kbd> (OR) 연산은 두 값이 모두 거짓일 때만 답이 거짓이 된다. 즉, 하나라도 참이면 답은 참이다.

```
(2 > 0) & (2 < 3)
# out[] : True
(2 > 2) | (2 < 3)
# out[] : True
```

### 숫자(Numbers)
숫자는 다시 정수형(int), 부동소수점형(float), 복소수형(complex)로 구분된다.

```py
a = 10 * 5  # 정수
type(a)
# out[] : int
b = 10.0 * 5  # 실수
type(b)
# out[] : float
```

다음과 같은 숫자 표기법도 있으니 익혀두도록 하자.

```py
123e2  # 123e2 = 123.0 x 100 = 12300.0
# out[] : 12300.0
123e-2  # 123e-2 = 123.0 x 0.01 = 1.23
# out[] : 1.23
```

### 문자열(String)
파이썬과 같은 프로그래밍 언어에서는 글자를 문자열(string)이라고 부른다. 파이썬에서 문자열을 만들 때는 따옴표를 사용한다. 따옴표에는 큰 따옴표와 작은 따옴표가 있으며 시작 따옴표와 종료 따옴표만 같으면 어느 것을 사용해도 상관없다. 

```py
print("Hello!")            #큰따옴표

print('한글도 쓸 수 있어요.') #작은따옴표

```

#### 문자열 연산
문자열도 숫자처럼 덧셈과 곱셈 연산을 할 수 있다. 덧셈 연산은 두 문자열을 붙이고 곱셈 연산은 문자열을 반복한다.
숫자를 문자열과 더하려면 `str` 명령을 써서 숫자를 문자열 자료형으로 바꾸어야 한다.

```py
print("내 이름은 " + "홍길동" + "입니다.")
# out[] : 내 이름은 홍길동입니다.

n = 10
print("별표를 " + str(n) + "번 출력합니다.")
print("*" * n)
# out[] : 별표를 10번 출력합니다.
# out[] : **********
```

### 여러 개의 자료를 한 변수에 담기 (list, dictionary, set)
파이썬에서는 하나의 변수에 여러 개의 자료를 한꺼번에 저장하고 필요한 때 꺼내 쓸 수도 있다. 파이썬에서 하나의 변수에 여러 개의 자료를 저장하고 쓰는 데는 다음의 방법을 사용한다.

* 여러 개의 자료가 순서가 있을 때는 **리스트(list)** 자료형 사용
* 여러 개의 자료가 이름이 있을 때는 **딕셔너리(dictionary)** 자료형 사용
* 여러 개의 자료가 순서도 없고, 중복도 없을 때는 **세트(set)** 자료형 사용

#### List
리스트 자료형에는 여러개의 자료를 순서에 따라 저장할 수 있다. 리스트 자료형 변수는 다음처럼 만든다.

```
리스트변수 = [자료1, 자료2, 자료3]
```

리스트 자료형 변수에서 특정한 자료를 꺼내려면 인덱싱(indexing)이라는 연산을 사용한다. 인덱싱 연산을 하려면 다음과 같이 대괄호(bracket)를 변수 이름 끝에 붙이고, 그 안에 자료의 순서를 뜻하는 숫자를 넣는다.
이 때 자료의 순서를 뜻하는 숫자를 인덱스(index)라고 한다. 파이썬에서는 가장 첫번째 자료를 가리키는 인덱스는 1이 아니라 0이다.

```py
x = [88, 90, 100]
x[0]
# out[] : 88

x[1]
# out[] : 90
```

리스트 변수에 또 다른 리스트 변수나 딕셔너리 변수를 넣을 수도 있다.

```py
e = [[1, 10], [2, 20]]
e[0]
# out[] : [1, 10]
```

#### Dictionary
딕셔너리 자료형에는 여러개의 자료를 이름을 붙여서 저장할 수 있다. 딕셔너리 자료형 변수는 다음과 같이 만든다.

```
딕셔너리변수 = {자료이름1: 자료값1, 자료이름2: 자료값2, 자료이름3: 자료값3}
```

파이썬에서는 자료의 이름을 키(key)라고 하고 자료의 값을 값(value)라고 한다.
딕셔너리 자료형 변수에서 특정한 자료를 꺼낼 때도 마찬가지로 인덱싱(indexing) 연산을 한다. 딕셔너리 자료형 변수에서 인덱싱 연산을 할 때도 다음처럼 대괄호(bracket)를 변수 이름 끝에 붙이고 그 안에 자료의 키를 넣는다.

```py
b = {"math": 88, "english": 90, "history": 100}
b["math"]
# out[] : 88
```

#### Set
집합(set)은 수학의 집합 개념과 동일한 자료형이다. 중복을 허용하지 않고 순서가 없는 데이터의 모임으로, 교집합, 합집합, 차집합 등의 집합 연산을 쉽게 처리할 수 있다.

```
s1 = set([1, 2, 3])
s2 = {1, 2, 3}
# s1, s3은 동일한 결과

s3 = set("Hello")
s3
# out[] : {'e', 'H', 'l', 'o'}
# 집합은 중복을 허용하지 않기 때문에 원소 'l'이 1개만 있는데 주목
```

### tuple
튜플(tuple)은 몇 가지 점을 제외하면 리스트와 거의 비슷하다. 리스트와 다른 점은 다음과 같다.

* 리스트는 [], 튜플은 ()으로 둘러싼다.
* 리스트는 요솟값의 생성, 삭제, 수정이 가능하지만, 튜플은 요솟값을 바꿀 수 없다.

## 제어문

### if 문
다음은 if와 else를 사용한 조건문의 기본 구조이다.
```
if 조건문:
    수행할_문장1 
    수행할_문장2
    ...
elif 조건문:
    수행할_문장1
    수행할_문장2
    ...
...
else:
   수행할_문장1
   수행할_문장2
   ... 
```
조건문 다음에 콜론 `(:)`을 잊지 말아야 한다.

다음은 if문의 예제이다.

```py
score = 85
if score >= 60:
   result = "합격"
else:
   result = "불합격"

print(result)
# out[] : 합격
```

### while 문
다음은 while 문의 기본 구조이다.
```
while 조건문:
    수행할_문장1
    수행할_문장2
    수행할_문장3
    ...
```
while 문은 조건문이 참인 동안 while 문에 속한 문장들을 반복해서 수행한다.
while 문 다음에 콜론 `(:)`을 잊지 말아야 한다.

1부터 10까지의 숫자 중에서 홀수만 출력하는 것을 while 문을 사용해서 작성해 보자.

```py
a = 0
while a < 10:
   a = a + 1
   if a % 2 == 0: continue
   print(a)
```

### for 문
for 문의 기본 구조는 다음과 같다.
```
for 변수 in 리스트(또는 튜플, 문자열):
    수행할_문장1
    수행할_문장2
    ...
```
for 문 다음에 콜론 `(:)`을 잊지 말아야 한다.

다음은 for문의 예제이다.

```py
test_list = ['one', 'two', 'three'] 
for i in test_list: 
   print(i)
```

## 파이썬 함수
파이썬에는 `def` 키워드와 `return` 키워드를 사용하여 다음과 같이 함수를 만들 수 있다. 

```
def 함수이름(입력변수이름):
   출력변수를 만드는 명령
   return 출력변수이름   
```
def 문 다음에 콜론 `(:)`을 잊지 말아야 한다.

다음은 파이썬 함수의 예제이다.

```py
def sum(a, b, c):
   s = a + b + c
   return s
sum(1, 2, 3)
# out[] : 6
```

## 공학용 파이썬 패키지
요리를 할 때 식재료를 각각 준비하여 재료손질부터 진행하기도 하지만 밀키트(meal kit)를 이용하면 실력이 부족한 사람도 그럴듯한 요리를 만들어 낸다.  프로그램에서도 전체 코드를 직접 만드는 것 보다는 여러가지 프로그램에 공통적으로 사용되는 code 부분을 해당 분야의 전문가가 미리 만들어 놓은 프로그램이 있으면 이를 가져다 사용하는 것이 현명한 선택이다. 이처럼 다른 프로그램 제작에 사용하기 위해 미리 만들어진 프로그램의 집합을 라이브러리(library)라고 한다. 파이썬에서는 라이브러리라는 용어 대신 패키지(package)라는 말을 주로 사용한다.

### 패키지 관리자
패키지를 설치하기 위한 프로그램을 패키지 관리자라고 한다. 파이썬에서는 pip라고 하는 패키지 관리자를 기본으로 제공하지만, 아나콘다 배포판에서는 conda라고 하는 아나콘다 전용 패키지 관리자도 제공한다. `conda`는 아나콘다 배포판이 설치될 때 자동으로 함께 설치된다.

### pip를 이용하여 패키지를 설치하기
pip 패키지 관리자를 이용하여 패키지를 설치하려면 콘솔 창에서 다음과 같이 입력한다.

```
pip install 패키지이름
```

예를 들어 quandl이라는 이름의 패키지를 설치하려면 ``pip install quandl`` 이라고 명령한다.

과학/공학 계산용으로 파이썬을 이용할 사람이라면 설치가 필요한 필수 패키지가 존재한다. 아래의 내용을 참조하여 미리 설치해 둘 필요가 있다.

```
pip install numpy
pip install scipy
pip install matplotlib
pip install seaborn
pip install scikit-learn
pip install sympy
pip install pandas
```

## 파이썬의 문자열 형식화
파이썬에서는 복잡한 문자열 출력을 위한 문자열 형식화(string formatting)를 지원한다. 문자열을 형식화하는 방법에는 `%` 기호를 사용한 방식과 `format` 메서드를 사용한 방식, 그리고 `f` 문자열을 사용하는 방식이 있다. 파이썬의 `print()`문에서 활용 가능하다.

### % 기호를 사용한 문자열 형식화
다음은 % 기호를 사용한 문자열 형식의 예이다.

```py
"내 이름은 %s입니다." % "홍길동"        #string
# out[] : '내 이름은 홍길동입니다.'
"나는 %d살 입니다." % 12               #int
# out[] : '나는 12살 입니다.'
"원주율의 값은 %f입니다." % 3.141592    #float
# out[] : '원주율의 값은 3.141592입니다.'
"%d 곱하기 %d은 %d이다." % (2, 3, 6)
# out[] : '2 곱하기 3은 6이다.'
```

문자열 형식화는 다음과 같이 고급 옵션을 지정할 수도 있다.

```py
"[%20s]" % "*"   # [와 ] 사이에 20칸의 공백이 있다.
# out[] : '[                   *]'
"[%-20s]" % "A"  # 20칸의 공백의 앞쪽에 A를 인쇄한다.
# out[] : '[A                   ]'
x = 1 / 3.0  # 값은 0.3333333....
"%.5f" % x   # 소숫점 아래 5자리까지만 인쇄한다.
# out[] : '0.33333'
```

### ``format`` 메서드를 사용한 문자열 형식화
``format`` 메서드를 사용하여 문자열을 형식화하는 방법도 있다. 이 때는 `%` 기호로 시작하는 형식지정 문자열 대신 `{}` 기호를 사용한다. 또한 자료형을 표시할 필요가 없다. 문자열 내에서 `{` 문자를 출력하고 싶을 때는 `{{`라는 글자를 사용한다.

```py
"내 이름은 {}입니다.".format("홍길동")
# out[] : '내 이름은 홍길동입니다.'
"내 이름은 {{{}}}입니다.".format("홍길동")
# out[] : '내 이름은 {홍길동}입니다.'
```

### f 문자열(f-string)을 사용한 문자열 형식화
파이썬 3.6부터는 f 문자열(f-string)이라는 것을 사용할 수 있다. f 문자열은 문자열의 앞에 f 글자를 붙인 문자열이다. f 문자열에서는 `{}` 안에 변수의 이름을 바로 사용할 수 있다.

```py
name = "홍길동"
age = 32
print(f"{name}의 나이는 {age}살이다.")
# out[] : `홍길동의 나이는 32살이다.`
```
